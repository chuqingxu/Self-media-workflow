## AI 装机师短视频

定位：

- 人设：AI 装机师
- 一句话介绍：以前帮人装电脑系统，现在帮老板装公司 AI 系统。
- 目标人群：二三线城市中小企业老板
- 风格：老板朋友型，真人口播优先
- 平台：抖音 + 微信视频号
- 引流：个人微信
- 承接：中转站 token / AI 使用入口，企业定制 AI 项目

常见用法：

```text
用 AI 装机师短视频 skill，围绕“老板买了 AI 会员为什么员工还是不用”生成抖音和视频号脚本。
```

```text
用 AI 装机师短视频 skill，在我的 AI 视频号内容文件夹里生成 7 天选题测试计划。
```
