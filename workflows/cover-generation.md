# image2 封面生成工作流

## 目标

把短视频选题或脚本转成可发布的爆款封面，而不是只给封面字。

## 输入

- 视频主题或口播脚本
- 目标平台: 抖音 / 视频号 / 小红书
- 是否真人出镜
- 可选: 用户头像、真人照片、封面样本、竞品封面、爆款拆解库条目

## 执行步骤

1. 提炼卖点: 老板痛点、冲突、结果、承接方向。
2. 写 3 组封面字: 主标题尽量短，重点字可用黄/青/绿突出。
3. 选封面方向: 强冲突、强结果、避坑提醒、工具对比、老板必看。
4. 生成 image2 提示词: 人物、姿态、背景、文字布局、颜色、禁用元素。
5. 调用 image2 模型能力或当前环境中等价的 image2-capable 生图入口，实际生成封面。
6. 检查生成图: 中文是否正确、人物是否自然、是否像“老初/AI 装机师”。
7. 必要时二次定稿: 用剪映、HyperFrames、Canva、Figma 或本地脚本叠加精确中文大字。
8. 做抖音/星图封面风险检查: 不出现加微信、二维码、手机号、第三方 App 跳转命令。

## 默认样式

- 真人近景或半身
- 黑 T、正面镜头、手指指向或拿手机
- 背景虚化，办公室/园区/暗科技感均可
- 大号中文标题覆盖画面上半部或中部
- 白色、青蓝、绿色、黄色重点字
- 黑色描边、厚阴影、3D 爆款感

## 输出

使用 `templates/cover-generation-template.md` 记录结果。

如果无法实际调用 image2 模型能力或等价图片生成入口，输出必须标注：

```text
Tool Execution: none
原因: 当前环境没有可用 image2/image generation 工具
```
